  • help - Provides the help page for Spacey Boi.

    Usage:

    • /help [command]

    Examples:

    • /help ban

  • music - Play some music (use again with no music selected to access menu).

    Usage:

    • /music [url] or [song name]

    Examples:

    • /music Cupid FIFTY FIFTY

  • invite - Creates 10 day invite for this server.

    Usage:

    • /invite

  • uptime - Returns uptime.

    Usage:

    • /uptime

  • userinfo - Displays userinfo about the selected user.

    Usage:

    • /userinfo [member]

    Example:

    • /userinfo Lonely#1111

  • serverinfo - Gets You information about the server.

    Usage:

    • /serverinfo

  • roleinfo - Displays role about the selected role.

    Usage:

    • /roleinfo [role]

    Example:

    • /roleinfo @Developer

  • channelstats - Fetches information about the channel where the cmd is used

    Usage:

    • /channelstats

  • mentions - Counts the amount of mentions You got in last certain amount of messages

    Usage:

    • /mention [limit] [user]

    Example:

    • /mention 100 Lonely#1111

  • warn list - Checks total warnings for a user.

    Usage:

    • /warn list [user]

    Example:

    • /warn list Lonely#1111

  • membercount - Counts total members.

    Usage:

    • /membercount

  • sys - Some info about the system.

    Usage:

    • /sys

  • poll - Creates a pool with 2 option

    Usage:

    • /poll [question] [a] [b]

    Example:

    • /poll Is the sky blue? Yes No

  • qrcode - Generate a QR code.

    Usage:

    • /qrcode [url]

    Example:

    • /qrcode https://spaceyboi.com/

  • ban - Bans a Member from the server.

    Usage:

    • /ban [member] [reason]

    Example:

    • /ban Lonely#1111 Too Good!

  • unban - Unbans a member from the server.

    Usage:

    • /unban [member] [reason]

    Example:

    • /unban Lonely#1111 Too Good!

  • kick - Kicks a user from the server.

    Usage:

    • /kick [member] [reason]

    Example:

    • /kick Lonely#1111 Too Good!

  • delete - Delete an amount of messages.

    Usage:

    • /delete [amount]

    Example:

    • /delete 100

  • timeout - Applies timeouts to a member.

    Usage:

    • /timeout [member] [minutes]

    Example:

    • /timeout Lonely#1111 60

  • mute - Mutes a member for a specific duration of time.

    Usage:

    • /mute [member] [time] [reason]

    Example:

    • /unmute Lonely#1111 60m Too Good!

  • unmute - Unmutes a member.

    Usage:

    • /unmute [member]

    Example:

    • /unmute Lonely#1111

  • slowmode - Changes channels slowmode.

    Usage:

    • /slowmode [seconds]

    Example:

    • /slowmode 5

  • nick - Change a members nickname.

    Usage:

    • /nick [member] [nickname]

    Example:

    • /nick Lonely#1111 NotLonely

  • lock - Locks the channel for the server members.

    Usage:

    • /lock

  • unlock - Unlocks the channel for the server members.

    Usage:

    • /unlock

  • nukechannel - Nukes a channel and re-create another clone of it.

    Usage:

    • /nukechannel

  • role - add or remove role from a User.

    Usage:

    • /role [member] [role]

    Example:

    • /role Lonely#1111 @Developer

  • warn add - Warns a member.

    Usage:

    • /warn add [member] [reason]

    Example:

    • /warn add Lonely#1111 Too Good!

  • warn remove - Removes a warning from a member.

    Usage:

    • /warn remove [warn_id]

    Example:

    • /warn remove obUlp2ZlMH4

  • warn list - Shows a list of warnings for a member.

    Usage:

    • /warn list [member]

    Example:

    • /warn remove Lonely#1111

  • autorole - Sets up auto role for your server.

    Usage:

    • /autorole [role]

    Example:

    • /autorole @Developer

  • autorolereset - Resets the servers auto role.

    Usage:

    • /autorolereset
